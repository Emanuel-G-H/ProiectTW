Cerință

Să se creeze o platformă Web ce ajută cu exersarea cunoștințelor de HTML și CSS. După autentificarea cu o rețea socială la alegere (GitHub, Facebook, Twitter, etc), aplicația va pune la dispoziția utilizatorilor, printr-o interfață atractivă, nivele cu diverse provocări, unde aceștia vor trebui să scrie cod pentru a trece mai departe.

Aplicația va salva progresul fiecărui utilizator, acesta putând continua de unde a rămas, mai târziu. În plus, aceasta va oferi statistici, referitoare la clasamentele globale cât și pentru fiecare utilizator în parte. De asemenea, platforma va expune și un API REST/GraphQL prin care alte aplicații vor putea prelua în format JSON sau XML atât provocarile, cât și răspunsurile corecte.

Introducere

Prioritar dezvoltarii platformei Web a fost atingerea scopului didactic pe care aceasta si-l propune. Pentru ca experienta utilizatorilor sa fie intr-adevar una placuta, interfata urmareste o tematica inspirata din lumea Lego , iar fiecare tip de nivel (HTML sau CSS) contine un scenariu Lego pe care utilizatorii trebuie sa il parcurga pentru a avansa atat in universul Lego, cat si in universul programarii. Interfața cu utilizatorul

Platforma Web ofera o experienta placuta utilizatorului, abordand o tema inspirata din lumea Lego( vezi mai multe detalii in Introducere) ce are scopul de a starni interesul dar si creativitatea. Prima pagina a platformei Web este pagina de autentificare ce contine un buton prin intermediul caruia un utilizator isi poate accesa contul. Autentificarea se face utilizand conturi GitHub iar dupa autentificare, utilizatorii sunt redirectionati catre pagina principala, de unde poate naviga ulterior in pagina de jocuri, pagina contului si pagina de statistici.

Interfata paginii de autentificare

Pagina de autentificare intampina utilizatorii cu titlul platformei web (LEGO HTML & CSS), o scurta descriere ce incurajeaza utilizatorii sa foloseasca platforma si un buton de autentificare prin Github ce permite utilizatorilor sa isi acceseze conturile.

In urma autentificarii, daca aceasta are loc cu succes, utilizatorul este redirectionat catre pagina Home a platformei web.

Interfata paginii Home

Pagina Home este pagina ce reprezinta punctul de legatura al tuturor paginilor oferite de platforma. Pentru a ajunge in pagina Home, utilizatorii trebuie sa se autentifice cu succes in pagina de autentificare. O data autentificati, acestia pot accesa pagina Home apasand click pe logo-ul platformei web( aflat in antet). Pagina Home este construita astfel incat experienta de navigare sa fie una cat mai placuta, cuprizand diverse sectiuni ce au rolul de a ajuta utilizatorii.

Interfata paginii Games

Pagina Games este o pagina statica ce contine doua subpagini dedicate nivelelor din HTML cat si CSS. Aceasta pagina este creata astfel incat sa se plieze pe dimensiunile oricarui dispozitiv ce acceseaza platforma. Astfel, pozitionarea nivelelor se face fie vertical, fie orizontal.

Interfata nivele

Pagina unui nivel, fie HTML, fie CSS, contine numarul nivelului, descrierea nivelului, un spatiu de completare si rezolvare al nivelului si o sectiune ce surprinde partea practica a nivelelor. In functie de rezolvarea utilizatorilor, acestia pot trece sau nu la nivele mai avansate. In cazul in care o rezolvare este corecta, platforma va puncta utilizatorul, va efectua modificarile din sectiunea practica-grafica, si va debloca urmatorul nivel.

Interfata paginii Statistics

Pagina Statistics este pagina ce gazduieste clasamentul utilizatorilor. Clasamentul acesta se face in functie de punctele castigate in urma efecturarii nivelelelor de pe pagina Games. Clasamentul este facut pentru fiecare sectiune:

  • Clasamentul general;
  • Clasamentul pentru sectiunea CSS;
  • Clasamentul pentru sectiunea HTML;

Pentru fiecare utilizator din clasament, se afiseaza locul, numele utilizatorului si punctajul acestuia.
